# Smart Diet Planner

This is a frontend web application that calculates BMI and provides diet suggestions based on user input.

## Features
- BMI Calculation
- Diet Suggestions for Weight Loss and Gain
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How it Works
User enters details like age, weight, height, and goal.
The system calculates BMI and provides diet suggestions.
